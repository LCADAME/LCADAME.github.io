Machine Learning Boosted PVT Solver

The main goal of this project is to develop a PVT solver specially for the case where a high percentage of CO2 is present in the mixture, capabale of handling 3-phase configurations (gas-CO2 liquid-liquid).

Another important aspect of this project is to use machine learning techniques to enhance the fidelity of the results, using available experimental data.
